## Overview
This repository contains a security audit report of the OWASP Juice Shop application, an intentionally vulnerable web application designed for security training.

The assessment was conducted in a controlled local environment strictly for educational purposes.

## Environment
- OS: Kali Linux
- Deployment: Docker
- Tools Used:
  - Burp Suite
  - Nmap
  - Firefox Browser

## Methodology
- Application reconnaissance
- Traffic interception and analysis
- Vulnerability testing
- Exploitation and confirmation
- Documentation and remediation

## Key Findings
- Broken Access Control (Scoreboard Access)
- SQL Injection (Authentication Bypass)
- Insecure Direct Object Reference (IDOR)
- Security Misconfiguration (CORS)
- Sensitive Data Exposure (JWT)
